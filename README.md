Interpretation of Statistics 
What we learn from the PCA: 
The main axis of morphometric variation is represented by PC1 (51%). The most significant characteristics that set people apart are those with large loadings on PC1 (X.5, X.7, and X.4). 
Independent variation orthogonal to PC1 is captured by PC2 (17.2%). This axis draws attention to various morphological features that differ from the primary pattern. 
Group Differences: The overlap (or separation) of confidence ellipses and the locations of group centroids show if there are notable morphometric differences between groups.
